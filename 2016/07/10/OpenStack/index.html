OpenStack

发表于 2016-07-10 | 分类于 Linux | 阅读次数

一些组件

keystone 用户认证和服务目录
RabbitMQ 消息队列
MySQL 提供数据存储

Nova-nova compute-libvirt
KVM虚拟机-qemu-kvm进程
nova-scheduler 调度，找可用的计算节点

glance 提供镜像 qcow2 raw
neutron 提供网络 FLAT VLAN GRE vxlan
cinder 提供硬盘（可选） cinder-scheduler cinder-volume

Openstack 私有云 用于开发 测试 生产，时间部署注意：
IO；不要改主机名；内网DNS；时间同步；弱口令；组件注册的IP绑定；熟悉virsh libvirt；桌面虚拟化

  • 基础服务：
    MySQL 3306
    RabbitMQ 5672 15672 25672
    Keystone （权限管理、服务目录） 5000 35357
    memcached 11211
  • 镜像服务：
    Glance glance-api 9292 glance-registry 9191
  • 计算服务：
    Nova api 8774 metadata 8775 Nova Scheduler 1.过滤 2.权值 novncproxy 6080
  • 网络服务：
    neutron

OSI七层：
传输层： LBaas Haproxy 4层和7层
网络层： L3-agent Linux静态路由+NAT
数据链路层： ML2 linuxbridge openvswith 商业插件
块存储：
cinder cinder-api cinder-scheduler cinder-volume
